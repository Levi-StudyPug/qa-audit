## 💳 How Anthropic API Billing Works

### Pay-As-You-Go Model
1. Add credits to account (minimum $5)
2. API calls deduct from balance
3. Charged per API request (~$0.15 per page)
4. No monthly fees or subscriptions

### Setting Up
1. Create account: https://console.anthropic.com/
2. Add payment method
3. Load credits (recommended: $10-20 for testing)
4. Create API key
5. Set spending limits (optional but recommended)

### Monitoring Usage
- Real-time balance in console
- Usage history and logs
- Set alerts for spending thresholds
- Monthly invoices

## ❓ FAQ

**Q: Can I audit without the API key?**  
A: No, the audit step requires Claude API access. But you can capture all screenshots for free and audit later.

**Q: Can I use a free AI instead?**  
A: Free LLMs (like local Llama models) won't provide the same quality of analysis. Claude Sonnet 4 is specifically trained for this type of structured evaluation.

**Q: What if I go over budget?**  
A: Set spending limits in Anthropic console. Once limit is reached, API stops working until you increase it.

**Q: Can I get a refund if quality is poor?**  
A: Test with 5-10 pages first ($0.75-$1.50) to verify quality before running full audit.

**Q: Is there a bulk discount?**  
A: Currently no volume discounts, but $0.15/page is already very competitive.

**Q: How often should I audit?**  
A: Capture screenshots weekly (free), audit monthly for changed pages, full quarterly audit.
